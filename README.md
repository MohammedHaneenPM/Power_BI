## Featured Dashboards

### Valorant Agent Performance Dashboard

**Objective:**
Analyze ranked match performance data to identify trends, agent effectiveness, and gameplay insights.

**Key Insights:**

* Compared win rates across agents
* Evaluated RR (Rank Rating) performance
* Analyzed kill-to-win relationships
* Identified high-performing and underperforming agents

* Data Modeling
* KPI Cards
* Interactive Filters
